# SnowBallChallenge
Brainstorm Snowball Challenge

Booker: 
En enkel chip man kan klistre på innsiden av permen på bøker, som lokaliserer boken. Når man er ferdig å lese en bok, kan man legge den fra seg på en cafe eller et annet offentlig sted å merke boken som ledig. Slik kan man gjennom en app finne forskjellige bøker rundt omkring i byen. Min 3 bøker per bruker. 
En åpen bo

FlexDoor: 
Enn app som kommuniserer med dører som kan automatisk åpne seg, slik at om man for eksempel sitter i rullestol og har appen innstalert vil døren åpne seg automatisk. 

Foodspire: 
En app hvor du scanner utløpsdato på varer du setter i kjøleskapet, og som deretter varsler deg om en vare er på vei til å gå ut på dato.
